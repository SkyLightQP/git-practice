# Git1

- It's version control system (VCS)
- Or using git when collaboration with peoples.

## .gitignore file

- You can ignore some file.

```
# .gitignore

.env
build/
```

## git config

- There are 3 type configs.
  - System Level
  - Global Level
  - Local Level

```
git config
```

## git init

- Initialize repository.

```
git init
```

## git add

- Add file to be staged
- It call be **tracked**

```
git add
```

## git rm

- Unstaging file.

```
git rm -cached [name]
```

## git comming

- Commit files in stage.
- You can include inline message using `-m` option.

```
git commit -m <message>
```

## git branch

- Change branch name

```
git branch -m <old> <new>
```
